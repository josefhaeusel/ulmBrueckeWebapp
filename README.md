# ulm-bruecke-frontend

## Deployment flachsbruecke-ulm.de (Hetzner Server)
### Server Commands
```
cd /var/www/ulmBrueckeWebapp/;
git pull;
npm run build
```

### Icecast Stream
```
ICECAST2 Port http://flachsbruecke-ulm.de:8000
```

## VueJS Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
